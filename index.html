🚀 Jenkins CI/CD Automation

Auto Deployment Project by Jerin Joseph

📌 Project Overview

This project demonstrates a complete CI/CD pipeline using Jenkins where code changes are automatically detected and deployed to a live server.

⚙️ Architecture

Code Push → GitHub → Jenkins → Build → Deploy → Live Website

🚀 What I Did

  • Connected GitHub with Jenkins using webhook
  • Created automated CI/CD pipeline
  • Configured Jenkins job for deployment
  • Hosted application using Nginx
  • Tested real-time auto deployment

⚡ How It Works

  • Push code to GitHub repository
  • Webhook triggers Jenkins automatically
  • Jenkins pulls latest code
  • Build and deployment executed
  • Live website updates instantly

🛠️ Tools Used

  • GitHub (Version Control)
  • Jenkins (CI/CD Automation)
  • Nginx (Web Server)
  • Linux (Deployment Server)
